## 本題

最近話題のZigをVScodeでセットアップした時の議事録です。

## 1.Zig本体をインストール

```bash
brew install zig
```

## 2.VScodeの拡張機能をインストール

私が入れているのは下記の拡張機能です。

https://marketplace.visualstudio.com/items?itemName=tiehuis.zig

https://marketplace.visualstudio.com/items?itemName=lorenzopirro.zig-snippets

https://marketplace.visualstudio.com/items?itemName=AugusteRame.zls-vscode

## 3.Zig Language Serverの設定

Zig Language Serverは自分でビルドします。
そのためにまずは `brew` でインストールした `zig` のバージョンを変えます。
下記から `master` のバージョンをインストールしてください。

https://ziglang.org/download/

その後下記コマンドでインストール場所を特定します。

```bash
which zig
```

大半の人が `/usr/local/bin/zig` だと思うので `/usr/local/bin/zig` で話を進めます。

`usr` フォルダはMacintosh HDの中に隠しフォルダとして存在しています。
ショートカットキー↓

```bash
[command] + [shift] + [.(ドット)]
```

`which` の結果を元に移動していき `zig` を右クリックしてオリジナルを表示します。
そこにある `zig` をインストールしてきた `zig` の `master` のバージョンに置き換えます。
下記コマンドで確認します。

```bash
zig version
```

その後好きなフォルダに移動して、公式のレポジトリのコマンドを実行してください。
rootをおすすめします。

https://github.com/zigtools/zls

```bash
git clone --recurse-submodules https://github.com/zigtools/zls
cd zls
zig build -Drelease-safe
./zig-out/bin/zls config # Configure ZLS
```

基本 `y` で答えていき、途中エディターを聞かれるので `vscode` と答えます。
その後 `VScode` で設定の検索窓に下記を入力します。

```text
@ext:AugusteRame.zls-vscode
```

下記の部分を設定します。
入力例は環境に合わせて変えてください。

```text
Zls:Build_runner_path

/Users/name/zls/zig-out/bin/build_runner.zig

Zls:Path

/Users/name/zls/zig-out/bin/zls

Zls:Zig_exe_path

/usr/local/bin/zig
```

## 4.完成

これでVScodeでZigが書けるようになったはずです。
シンタックスやエラーも表示されます。
